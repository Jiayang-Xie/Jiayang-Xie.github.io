    paste(): Takes multiple elements from the multiple vectors and concatenates them into a single element.
    __paste()__：从多个向量中获取多个元素，并将它们连接为一个元素
    paste0(): The paste0() function has space as its default separator and limits your opportunities in the output as well.
    __paste0()__： paste0（）函数使用空格作为其默认分隔符。

  • 语法

x = vector having values.
x = 具有值的向量。

sep = separator symbols that can be used to separate the elements.
sep =可用于分隔元素的分隔符。paste中默认为空格，paste0中为空字符，sep =””

collapse = It gives a value to collapse.
The separator will deal with the values which are to be placed in between the set of elements and the collapse argument will make use of specific value to concatenate the elements into single -string.

总结， sep处理将要放置在元素集之间的值，而collapse将使用特定值将元素连接为单个字符串。

建立博客的动机

我的硕士课题是关于神经干细胞的研究，其中一大半涉及到了生物信息学主要是转录组的分析。当然这也是我有心下的选择。而近几年的疫情尤其是上海最近的疫情，使得我须向此方向重重发力。其实在大概一年前就开始了笼统的RNA-seq学习，但没有养成类似湿实验的实验记录习惯，导致学习不牢固，很多东西或者工具过一会就忘记了，待到用时又得回过头在学一遍，费时费力，事倍功半。

在此背景下，便萌生了记录自己课题学习过程的想法。但此类记录涉及许多链接和代码，常用的OneNote不太适用，便需要寻找适用的记录平台。整好在学习过程中看到了别人个人博客的技术贴分享，觉得不错，便也打算建立自己的个人博客。以上就是动机来源了。

建立博客的过程

大体流程

自我走了一遍的经验来看，整个流程可以分为基于nodejs搭建的hexo博客框架，框架搭好后就是页面配置（configuration），这一步大部分都是套别人做好的主题（theme），类似手机的主题壁纸。到这里基本上就可以完成一个写作记录的闭环了。只不过这个闭环是不与网络连接的，要在互联网上留存，就要做成个人网以供别人访问。这里有两种方式，一种免费的是利用GitHub Pages创建仓库(repository)；另一种就是花钱买一个服务器将网站上部署到上面去，当然其中也需要Git的安装部署。除此之外，还需要给你的网站买一个域名。这样个人的博客网站就部署完成了，当然其中的流程也挺长的。
